Python 是一种动态类型语言，在编写代码时无需显式声明变量类型，解释器会在运行时自动推断。Python 的数据类型主要分为标量类型（单个值）和容器类型（多个值的集合）。以下是 Python 核心基础数据类型的系统梳理：

### 一、 数值类型 (Numeric Types)
1. **整数（int）**：表示没有小数部分的数字，可以是正数、负数或零。在 Python 3 中，整数没有大小限制（受限于内存），且统一为 `int` 类型，不再区分长整型。支持十进制、二进制（`0b`）、八进制（`0o`）和十六进制（`0x`）表示。
2. **浮点数（float）**：表示带有小数部分的实数，支持科学计数法（如 `1.2e3`）。需要注意的是，浮点数基于 IEEE 754 标准，在运算时可能存在精度误差（如 `0.1 + 0.2 != 0.3`），高精度计算可通过 `decimal` 模块解决。
3. **复数（complex）**：由实部和虚部组成，虚部以 `j` 或 `J` 标记，例如 `3+4j`。

### 二、 布尔类型 (Boolean Type)
**布尔值（bool）**：用于表示逻辑真值，仅有 `True` 和 `False` 两个取值。本质上它是整数的子类，`True` 等价于 1，`False` 等价于 0，因此可以参与算术运算。在 Python 中，任何“空”或“零”值（如 `0`、`""`、`[]`）转换为布尔值时均为 `False`，其余为 `True`。

### 三、 序列类型 (Sequence Types)
1. **字符串（str）**：由 Unicode 字符组成的不可变序列，用于存储文本信息。支持单引号、双引号和三引号定义，可进行索引、切片、拼接等操作。
2. **列表（list）**：可变的有序序列，使用方括号 `[]` 定义。列表可以包含任意类型的数据对象，并支持动态添加、删除或修改元素。
3. **元组（tuple）**：不可变的有序序列，使用圆括号 `()` 定义。一旦创建便不能修改其内容，由于不可变特性，元组可以作为字典的键。
4. **范围（range）**：表示不可变的数字序列，通常用于 `for` 循环中。

### 四、 映射类型 (Mapping Type)
**字典（dict）**：可变的键值对集合，使用花括号 `{}` 定义。字典是无序的，其中的“键”必须是唯一且不可变的类型（如字符串、元组等），而“值”可以是任意数据类型。

### 五、 集合类型 (Set Types)
1. **集合（set）**：可变、无序且元素不重复的集合，使用花括号 `{}` 定义。常用于数据去重以及数学意义上的集合运算。
2. **冻结集合（frozenset）**：与集合类似，但属于不可变类型，一旦创建就不能被修改。

### 六、 二进制类型 (Binary Types)
1. **字节（bytes）**：不可变的字节序列，通常以 `b"..."` 的形式表示，用于存储二进制数据。
2. **字节数组（bytearray）**：与 bytes 类似，但是可变的。

### 七、 特殊类型
**NoneType**：该类型只有一个值 `None`，常用于表示空值、未定义值或函数没有返回值的情况。

💡 **进阶理解：可变性 vs 不可变性**
在 Python 中，区分数据类型的一个核心维度是“是否可变”。数值类型、字符串、元组和冻结集合是**不可变（Immutable）**的，这意味着它们的内容一旦创建就无法更改；而列表、字典和普通集合是**可变（Mutable）**的，允许随时增删改内部元素。你可以使用内置的 `type()` 函数来随时检查变量的具体数据类型。
